# Unidad 02 - Estructuras de Datos y Manejo de Errores

## Contenido

### Ejercicios Básicos

- **contar_vocales.py**: Cuenta las vocales en una cadena de texto
- **crear_lista.py**: Crea y manipula listas de números

### Algoritmos

- **basta.py**: Implementación del juego "Basta"
- **comparar_cantidades.py**: Compara valores numéricos
- **cuenta_atras.py**: Implementa una cuenta regresiva
- **nota_media.py**: Calcula el promedio de calificaciones

### Manejo de Archivos

- **escritura_archivo.py**: Escribe datos en archivos con manejo de excepciones
- **try-except_2.py**: Ejercicios de manejo de errores
- **try-except_3.py**: Ejercicios avanzados de excepciones

## Conceptos Clave

- Listas y operaciones con listas
- Cadenas de texto y métodos de string
- Manejo de excepciones (try-except-finally)
- Lectura y escritura de archivos
- Bucles for y while
- Comprensiones de listas
